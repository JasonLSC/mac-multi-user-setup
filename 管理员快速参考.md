# 管理员快速参考

## 🚀 创建新用户

### 使用简化脚本（推荐）

```bash
cd /Users/lisicheng/mac-multi-user-setup

# 基本用法（使用默认密码 user2026）
sudo ./scripts/quick-create-user.sh 用户名

# 指定密码
sudo ./scripts/quick-create-user.sh 用户名 自定义密码

# 指定密码和 API key
sudo ./scripts/quick-create-user.sh 用户名 密码 sk-ant-api03-xxxxx
```

### 示例

```bash
# 创建用户 zhangsan，使用默认密码
sudo ./scripts/quick-create-user.sh zhangsan

# 创建用户 lisi，密码为 lisi123
sudo ./scripts/quick-create-user.sh lisi lisi123

# 创建用户 wangwu，带密码和 API key
sudo ./scripts/quick-create-user.sh wangwu ww2026 sk-ant-api03-xxxxx
```

## 🔧 常用管理命令

### 查看所有用户

```bash
dscl . list /Users | grep -v "^_"
```

### 查看用户信息

```bash
id 用户名
```

### 重置用户密码

```bash
sudo dscl . -passwd /Users/用户名 新密码
```

### 删除用户

```bash
# 删除用户（保留主目录）
sudo dscl . -delete /Users/用户名

# 删除用户和主目录
sudo dscl . -delete /Users/用户名
sudo rm -rf /Users/用户名
```

### 查看登录用户

```bash
who
# 或
w
```

### 查看 SSH 连接

```bash
sudo lsof -i :22
```

## 🌐 SSH 管理

### 启用 SSH

```bash
sudo systemsetup -setremotelogin on
```

### 禁用 SSH

```bash
sudo systemsetup -setremotelogin off
```

### 查看 SSH 状态

```bash
sudo systemsetup -getremotelogin
```

### 查看 SSH 日志

```bash

```

### 踢出用户

```bash

sudo pkill -9 -t pts/0
```

## 🔒 安全管理

### 查看登录历史

```bash
last | head -20
```

### 查看失败的登录尝试

```bash
log show --predicate 'eventMessage contains "Failed"' --last 1d | grep ssh
```

### 限制 SSH 访问用户

编辑 SSH 配置：
```bash
sudo nano /etc/ssh/sshd_config.d/custom.conf
```

添加：
```
AllowUsers user1 user2 user3
```

重启 SSH：
```bash
sudo launchctl unload /System/Library/LaunchDaemons/com.openssh.sshd.plist
sudo launchctl load -w /System/Library/LaunchDaemons/com.openssh.sshd.plist
```

## 📦 批量创建用户

### 创建用户列表文件

```bash
nano users.txt
```

内容格式：
```
zhangsan:pass123:sk-ant-api03-key1
lisi:pass456:sk-ant-api03-key2
wangwu:pass789:sk-ant-api03-key3
```

### 批量创建

```bash
while IFS=: read -r username password apikey; do
    sudo ./scripts/quick-create-user.sh "$username" "$password" "$apikey"
done < users.txt
```

## 🆘 故障排除

### SSH 无法连接

```bash
# 1. 检查 SSH 服务
sudo launchctl list | grep ssh

# 2. 检查端口
sudo lsof -i :22

# 3. 重启 SSH 服务
sudo launchctl unload /System/Library/LaunchDaemons/com.openssh.sshd.plist
sudo launchctl load -w /System/Library/LaunchDaemons/com.openssh.sshd.plist

# 4. 查看日志
log show --predicate 'process == "sshd"' --last 10m
```

### 用户无法登录

```bash
# 检查用户是否存在
id 用户名

# 检查用户是否在 SSH 访问组
dscl . -read /Users/用户名 GroupMembership

# 重置密码
sudo dscl . -passwd /Users/用户名 新密码
```

### Claude Code 无法启动

```bash
# 切换到用户
sudo su - 用户名

# 检查 Claude 是否安装
which claude

# 检查配置
cat ~/.claude/config.json

# 重新配置
claude config
```
