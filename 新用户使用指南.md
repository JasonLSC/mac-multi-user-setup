## 🚀 快速开始

### 1. 首次连接

打开终端（Terminal），输入：

**本地网络连接：**
```bash
ssh 你的用户名@192.168.10.118
```

**外网连接：**
```bash
ssh -p 2222 你的用户名@188.253.4.121
```

输入密码后即可登录。

**建议首次登录后修改密码：**
```bash
passwd
```

### 2. 启动 Claude Code

登录后，按以下步骤操作：

```bash
# 1. 启动 tmux 会话（保持后台运行）
tmux new -s main

# 2. 配置 Claude Code API key（首次使用）
claude config
# 按提示输入你的 API key

# 3. 启动 Claude Code
claude

# 4. 开始使用！
```

### 3. 使用 OpenClaw（个人 AI 助手）

OpenClaw 是一个本地运行的 AI 助手，支持多种通讯渠道：

```bash
# 首次配置
openclaw onboard --install-daemon

# 启动网关
openclaw gateway --port 18789

# 发送消息测试
openclaw message send --to +1234567890 --message "Hello"

# 与助手对话
openclaw agent --message "你好" --thinking high
```

更多信息：https://openclaw.ai

### 4. 分离和重连会话

**分离会话（保持 Claude 在后台运行）：**
- 按 `Ctrl-a`，然后按 `d`

**重新连接到会话：**
```bash
ssh 你的用户名@192.168.10.118
tmux attach -t main
```

## 💡 常用命令

### Claude Code 快捷命令

```bash
cc          # 启动 Claude Code
ccc         # 继续上次会话
ccr         # 重置会话
```

### Tmux 会话管理

```bash
tmux new -s 会话名      # 创建新会话
tmux attach -t 会话名   # 连接到会话
tmux list-sessions     # 列出所有会话
tmux kill-session -t 会话名  # 关闭会话
```

或使用别名：
```bash
tn main     # 创建名为 main 的会话
ta main     # 连接到 main 会话
tl          # 列出所有会话
```

### Tmux 快捷键

在 tmux 会话中：

- `Ctrl-a |` - 垂直分屏
- `Ctrl-a -` - 水平分屏
- `Ctrl-a d` - 分离会话
- `Alt + 方向键` - 切换面板
- `Ctrl-a r` - 重新加载配置

## 🌐 远程访问

### 本地网络访问

```bash

```

### 外网访问（如果已配置）

```bash
ssh -p 2222 你的用户名@服务器公网IP
```

### 使用 SSH 密钥（推荐，更安全）

**在你的电脑上生成密钥：**
```bash
ssh-keygen -t ed25519
```

**复制公钥到服务器：**
```bash
ssh-copy-id 你的用户名@192.168.10.118
```

之后连接就不需要输入密码了。

## 💻 工作流程示例

### 典型的工作流程

```bash
# 1. 连接到服务器
ssh 你的用户名@192.168.10.118

# 2. 连接到你的工作会话（或创建新的）
tmux attach -t main || tmux new -s main

# 3. 启动 Claude Code
claude

# 4. 开始工作...

# 5. 完成后分离会话（Ctrl-a d）
# Claude 会继续在后台运行

# 6. 断开 SSH 连接
exit
```

### 多任务工作

```bash

tmux new -s project1
tmux new -s project2
tmux new -s testing

# 在会话间切换
tmux attach -t project1
tmux attach -t project2
```

## 🆘 常见问题

### Q: 忘记密码怎么办？
A: 联系管理员重置密码。

### Q: Claude Code 提示 API key 错误？
A: 运行 `claude config` 重新配置 API key。

### Q: 如何查看我的会话？
A: 运行 `tmux list-sessions` 或 `tl`。

### Q: 连接断开后我的工作会丢失吗？
A: 不会！只要你在 tmux 会话中工作，重新连接后可以继续。

### Q: 如何退出 Claude Code？
A: 输入 `exit` 或按 `Ctrl-d`。

### Q: 如何完全退出并关闭会话？
A: 在 tmux 会话中输入 `exit`，然后再次输入 `exit`。

### Q: 终端显示乱码怎么办？
A: 运行 `reset` 命令重置终端。

## 🔒 安全建议

1. **修改初始密码** - 首次登录后立即修改
2. **使用 SSH 密钥** - 比密码更安全
3. **不要分享密码** - 每个用户有独立账户
4. **定期更新密码** - 建议每 3 个月更新一次
5. **退出时分离会话** - 使用 `Ctrl-a d` 而不是直接关闭终端
